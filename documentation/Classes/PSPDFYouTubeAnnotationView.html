PSPDFYouTubeAnnotationView Class Reference

Inherits from UIView
Conforms to PSPDFAnnotationView
Declared in PSPDFYouTubeAnnotationView.h

Overview

Uses MPMoviePlayerController whenever possible, else falls back to the UIWebView YouTube plugin. Note: The YouTube plugin doesn’t show up in the Simulator. Test on the device!

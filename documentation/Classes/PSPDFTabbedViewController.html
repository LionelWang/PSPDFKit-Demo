PSPDFTabbedViewController Class Reference

Inherits from PSPDFBaseViewController : UIViewController
Declared in PSPDFTabbedViewController.h

Overview

Allows displaying multiple PSPDFDocument’s, easily switchable via a top tab bar. iOS5 only.

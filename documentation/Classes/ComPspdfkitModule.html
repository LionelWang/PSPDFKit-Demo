Overview

PSPDFKit Titanium Bridge.

Tasks

  • – PSPDFKitVersion

    Get the version of PSPDFKit. (This is not the version of the module bridge)
  • – showPDFAnimated:

    Shows a modal page with the pdf. At least one argument (the pdf name) is needed. Argument 2 sets animated to true or false. Arumgent 3 can be an array with options for PSPDFViewController. See http://pspdfkit.com/documentation.html for details. You need to write the numeric equivalent for enumeration values (e.g. PSPDFPageModeDouble has the numeric value of 1) Argument 4 can be an array with options for PSPDFDocument. Returns a PSPDFViewController object.
  • – clearCache:

    Clears the whole application cache. No arguments needed.
  • – setLanguageDictionary:

    Allows setting custom language additions. Accepts a dictionary with language dictionaries mapped to the language string.
  • – setLogLevel:

    Set internal log level. Defaults to 1 (PSPDFLogLevelError). PSPDFLogLevelNothing(0), PSPDFLogLevelError(1), PSPDFLogLevelWarning(2), PSPDFLogLevelInfo(3), PSPDFLogLevelVerbose(4).

PSPDFGlobalLock Class Reference

Inherits from NSObject
Declared in PSPDFGlobalLock.h

Overview

pdf reading needs memory, which is a rare resource. So we lock access very carefully.

Tasks

  • + sharedPSPDFGlobalLock

    get global singleton.
  • – tryLockWithPath:page:

    TRY to lock with path and real pdf page number (starts at 1). Get page via [PSPDFDocument pageNumberForPage:]. returns nil if currently locked.

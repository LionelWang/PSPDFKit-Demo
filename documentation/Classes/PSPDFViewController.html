PSPDFViewController Class Reference

Inherits from PSPDFBaseViewController : UIViewController
Conforms to UIPopoverControllerDelegate
UIScrollViewDelegate
Declared in PSPDFViewController.h

Overview

The main view controller to display pdfs. Can be displayed in fullscreen or embedded. When embedded, be sure to correctly relay the viewController calls of viewWillAppear/etc. (or use iOS5 view controller containment)

  •   overrideClassNames

    Use this to use specific subclass names instead of the default PSPDF* classes. e.g. add an entry of “PSPDFScrollView” / “MyCustomPSPDFScrollViewSubclass” as key/value pair to use the custom subclass. This replaces the previous properties “scrollViewClass” and “scrobbleBarClass” and adds support for many used classes.

         property
  •   toolbarEnabled

    Enables default header toolbar. Only displayed if inside UINavigationController. Defaults to YES. Set before loading view.

         property
  •   pageCurlEnabled

    Enables iBooks-like page curl feature. Works only with iOS5 or later. Falls back to default scrolling on iOS4. Since PSPDFKit 1.9. Note: doesn’t work well with non-equal sized documents. Use scrolling if you have such complex documents.

         property
  •   scrollingEnabled

    Enable/disable scrolling. can be used in special cases where scrolling is turned of (temporarily). Defaults to YES.

         property

    Status bar styling. Defaults to PSPDFStatusBarSmartBlack. If controller is used embedded (in a non-fullscreen way), this setting has no effect.

         property
  •   scrollOnTapPageEndEnabled

         property
  •   thumbnailSize

    Change thumbnail size. Default is 170x220.

         property

         property

    View that is displayed as HUD. Make a KVO on viewMode if you build a different HUD for thumbnails view. HUD is created in viewDidLoad. Subclass or use KVO to add your custom views when this changes.

         property

PSPDFScrollView Class Reference

Inherits from UIScrollView
Conforms to UIScrollViewDelegate
Declared in PSPDFScrollView.h

Overview

Scrollview for a single page. Every PSPDFPageView is embedded in a PSPDFScrollView.
